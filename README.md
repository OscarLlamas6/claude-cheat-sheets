## Activar GitHub Pages

1. Ir a **Settings → Pages** en el repo
2. Source: **Deploy from a branch**
3. Branch: `main` / `/ (root)`
4. Guardar — el sitio estará disponible en `https://oscarllamas6.github.io/claude-cheat-sheets`

## Agregar nuevos cheat sheets

1. Copiar el HTML a `sheets/`
2. Agregar una nueva `<a class="card">` en `index.html`
