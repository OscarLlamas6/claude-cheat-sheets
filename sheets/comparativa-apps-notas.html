🗂️ Comparativa de Apps de Notas y Gestión de Conocimiento (PKM)

Cuadro comparativo de 12 herramientas populares para tomar notas y construir una base de conocimiento personal: qué son, qué hacen, qué NO hacen, su modelo de negocio y cuándo (no) conviene usarlas. Usa el buscador para filtrar por nombre o característica.

Obsidian
Freemium (núcleo gratis) 		Editor de notas en Markdown local, orientado a "segundo cerebro" con vínculos bidireccionales y grafo de conexiones.
  • Grafo de relaciones entre notas
  • Enorme ecosistema de plugins (1000+)
  • Total control local, sin cuenta obligatoria
  • Personalización extrema (temas, CSS)
  • El core no es open source (cerrado, gratis para uso personal)
  • Sync y Publish son de paga
  • Colaboración en tiempo real limitada
  • Curva de aprendizaje con plugins
 Gratis para uso personal. Sync $8/mes, Publish $10/mes. Uso comercial requiere licencia. Win, macOS, Linux, iOS, Android Archivos .md locales Quieres control total de tus archivos y un ecosistema de plugins maduro. Necesitas open source real o colaboración en equipo fluida.
Joplin
Open Source 		App de notas Markdown 100% open source, con cifrado end-to-end y múltiples opciones de sincronización.
  • Totalmente gratis y open source (MIT)
  • Cifrado E2E real
  • Sync con Dropbox, Nextcloud, WebDAV, S3, etc.
  • Plugins y clipper de navegador
  • Grafo de conexiones muy básico o inexistente
  • Interfaz menos pulida que Obsidian/Notion
  • Colaboración en tiempo real inexistente
 100% gratis. Joplin Cloud (sync oficial) desde ~$2.99/mes opcional. Win, macOS, Linux, iOS, Android, Web Archivos .md + JEX (export propio) Buscas una alternativa gratuita, privada y verdaderamente open source a Evernote. Buscas vínculos bidireccionales avanzados o diseño visual moderno.
Blinko
Open Source 		Nota rápida + microblogging + IA (RAG) autoalojada. Mezcla capturas rápidas ("Blinkos") con notas detalladas y búsqueda semántica local.
  • Captura ultra rápida ("Quick Capture Box")
  • Búsqueda con IA/RAG en lenguaje natural (Ollama, OpenAI, etc.)
  • Self-hosted, control total de datos
  • Bot de Telegram y extensión de navegador
  • Requiere Docker/servidor propio (no trivial para no técnicos)
  • Apps móviles nativas más limitadas que Notion/Evernote
  • No apto para gestión de proyectos compleja
  • Comunidad y ecosistema aún pequeños (proyecto joven)
 Núcleo 100% gratis (GPL-3.0). Membresías opcionales $5–$9.90/mes por extras (extensión, bots, créditos IA). Web (self-hosted), macOS, Windows, Linux, Android Markdown + base PostgreSQL propia Quieres capturar ideas al vuelo con IA local, sin depender de la nube. No quieres mantener tu propio servidor/Docker.
Notion
Freemium 		Workspace todo-en-uno: notas, bases de datos, wikis y gestión de proyectos en un solo lugar en la nube.
  • Bases de datos relacionales muy flexibles
  • Colaboración en tiempo real excelente
  • Plantillas para casi cualquier caso de uso
  • IA integrada (Notion AI)
  • No es open source ni self-hosted
  • Puede volverse lento con bases grandes
  • Datos ligados a su nube (exportar es limitado)
  • Offline limitado
 Gratis (limitado) / Plus ~$10 / Business ~$20 por usuario/mes. Web, Win, macOS, iOS, Android Propietario (exporta a Markdown/CSV/PDF) Trabajas en equipo y necesitas wikis + proyectos + docs en un solo lugar. Te preocupa la privacidad de datos o quieres trabajar 100% offline.
Logseq
Open Source 		App de notas tipo "outliner" (basada en bloques) con enfoque en pensamiento no lineal, similar a Roam Research.
  • Estructura por bloques/outliner muy potente
  • Notas diarias (daily notes) nativas
  • Grafo de conexiones y backlinks
  • 100% local, código abierto
  • Rendimiento se degrada con grafos muy grandes
  • Interfaz menos amigable para principiantes
  • Sync oficial aún en evolución
  • Móvil menos maduro
 Completamente gratis. Plan Sync opcional de paga. Win, macOS, Linux, iOS, Android Markdown u Org-mode locales Prefieres pensar en bloques/outliner y quieres algo abierto y local. Manejas miles de notas y priorizas velocidad sobre estructura.
Anytype
Open Source (parcial) 		Espacio de trabajo local-first tipo Notion, con objetos, bases de datos y sincronización P2P cifrada.
  • Local-first con cifrado y sync P2P
  • Bases de datos y "objetos" muy flexibles
  • Sin necesidad de servidor propio para sincronizar
  • Diseño visual atractivo
  • Aún en desarrollo activo (algunas funciones inestables)
  • Solo parte del código es abierto
  • Ecosistema de plugins pequeño comparado con Obsidian
  • Curva de aprendizaje por su modelo de "objetos"
 Gratis con límite de almacenamiento en nube; planes de pago para más espacio/sync. Win, macOS, Linux, iOS, Android Formato propio (exportable a Markdown) Quieres algo tipo Notion pero local-first y más privado. Necesitas estabilidad total y un producto ya maduro.
Standard Notes
Open Source 		Notas simples, minimalistas, con cifrado end-to-end como prioridad número uno.
  • Cifrado E2E de extremo a extremo por defecto
  • Interfaz simple y estable, enfocada en durabilidad
  • Historial de versiones robusto
  • Editor extensible (Markdown, código, tablero, etc.)
  • Funciones avanzadas (editores extra, sync extendido) requieren plan pago
  • No tiene grafo de conexiones ni vínculos bidireccionales
  • Menos plugins que Obsidian
 Núcleo gratis. Plan "Productivity" ~$8.33/mes (facturado anual) para extras. Web, Win, macOS, Linux, iOS, Android Texto plano cifrado, exportable La privacidad y simplicidad a largo plazo son tu prioridad #1. Buscas un sistema tipo "segundo cerebro" con vínculos y grafo.
Trilium Notes
Open Source 		Editor jerárquico de notas (árbol) autoalojado, orientado a grandes bases de conocimiento personal.
  • Jerarquía de notas ilimitada tipo árbol
  • Cifrado de notas individuales
  • Scripting/automatización avanzada (JS)
  • Self-hosted, control total
  • Interfaz anticuada comparada con apps modernas
  • Requiere algo de configuración técnica para sync propio
  • Comunidad más pequeña
 100% gratis, sin planes de pago (self-hosted). Win, macOS, Linux, Web (server), Android (beta) Base de datos propia (exportable) Necesitas jerarquías profundas y automatización con scripts. Priorizas una interfaz moderna y pulida "out of the box".
Notesnook
Open Source 		Alternativa privada a Evernote, con cifrado end-to-end de código abierto y sync multiplataforma.
  • Cifrado E2E en todos los planes, incluso gratis
  • Sincronización fluida entre dispositivos
  • Interfaz moderna y amigable
  • Notebooks, tags y organización simple
  • Sin vínculos bidireccionales ni grafo
  • Adjuntos y almacenamiento limitados en plan gratis
  • Ecosistema de extensiones pequeño
 Gratis con límites. Plan Pro ~$4.49/mes. Web, Win, macOS, Linux, iOS, Android Formato propio cifrado (exportable) Quieres una alternativa simple y privada a Evernote/OneNote. Necesitas PKM avanzado con enlaces entre ideas.
AFFiNE
Open Source 		Editor híbrido documento/whiteboard/base de datos, pensado como alternativa open source a Notion + Miro.
  • Combina documento, pizarra infinita y base de datos
  • Local-first con opción de sync en la nube
  • Bloques editables tipo Notion
  • Código abierto activo y en crecimiento
  • Aún en beta/desarrollo activo, con bugs ocasionales
  • Rendimiento variable con documentos grandes
  • Ecosistema de plugins limitado
 Gratis (self-hosted u online). Plan Cloud Pro de pago para más almacenamiento. Web, Win, macOS, Linux, iOS, Android Formato propio (exportable a Markdown) Quieres una alternativa abierta a Notion con pizarra visual incluida. Necesitas estabilidad de nivel producción hoy mismo.
SiYuan
Open Source 		App de notas basada en bloques (block-based), con grafo de conocimiento y fuerte soporte para self-hosting.
  • Edición por bloques con referencias entre bloques
  • Grafo de relaciones y "flashcards" para repaso espaciado
  • Totalmente local y self-hosted
  • Exportación flexible (Word, PDF, etc.)
  • Interfaz y documentación menos accesibles para hispanohablantes/angloparlantes (origen chino)
  • Comunidad occidental más pequeña
  • Curva de aprendizaje moderada
 Gratis y open source; sync en la nube de pago opcional. Win, macOS, Linux, iOS, Android, Web Base de datos propia + exportable a Markdown Quieres repetición espaciada (flashcards) integrada a tus notas. Prefieres un ecosistema con más soporte y comunidad en español/inglés.
RemNote
Freemium 		Outliner de notas enfocado en estudio y memorización, con flashcards de repetición espaciada integradas.
  • Flashcards automáticas desde tus notas (spaced repetition)
  • Estructura tipo outliner con jerarquías
  • Ideal para estudiantes y oposiciones
  • Vínculos entre conceptos
  • No es open source
  • Funciones clave (más flashcards, IA) tras muro de pago
  • Menos enfocado en "segundo cerebro" general que en estudio
 Gratis limitado. Plan Pro ~$8/mes (anual). Web, Win, macOS, iOS, Android Propietario (exportable a Markdown) Estudias con frecuencia y quieres flashcards nativas desde tus apuntes. Buscas privacidad total u open source.
Tana
Freemium 		Workspace "AI-native" basado en un supertag system: mezcla notas, bases de datos y automatización con IA.
  • Sistema de "supertags" muy potente para estructurar datos
  • IA integrada para resumir y generar contenido
  • Búsquedas/consultas dinámicas tipo base de datos
  • Colaboración en equipo
  • No es open source ni self-hosted
  • Curva de aprendizaje alta (conceptos nuevos)
  • App móvil y offline aún limitados
  • Precio más alto que la mayoría de alternativas
 Gratis limitado. Plan de pago desde ~$10-20/mes por usuario. Web, Win, macOS (apps de escritorio vía navegador), iOS, Android Propietario (exportable parcialmente) Quieres combinar notas + bases de datos + IA en flujos avanzados. Buscas algo simple, gratuito o completamente offline.

Notas sobre esta comparativa: Los precios y características cambian con frecuencia; verifica siempre en el sitio oficial de cada herramienta antes de decidir. "Open Source" indica que el código fuente del núcleo de la app es público y auditable (aunque el hosting/soporte pueda tener planes de pago); "Freemium" indica que el software base es gratis pero con funciones o límites que requieren pago para desbloquearse; "Código cerrado" indica que el software no es open source aunque tenga un plan gratuito.
