Visión por Computadora · Guía de herramientas

10 herramientas para anotar datos y entrenar modelos de Computer Vision

Casi todo proyecto de visión por computadora empieza por lo mismo: conseguir datos etiquetados. Estas son las herramientas más usadas para anotar imágenes, automatizar el etiquetado y, en algunos casos, entrenar y desplegar el modelo — qué son, qué no hacen, si son gratis y cuándo conviene (o no) usarlas.

00 Cómo leer este cuadro

Cada herramienta lleva una etiqueta de costo. Los precios cambian seguido: úsalos como orientación y confirma siempre en la web oficial antes de decidir.

Open source

Código abierto y gratis. Puedes usarlo sin pagar; algunas ofrecen además una versión de nube de pago.

Freemium

Tiene un plan gratuito real (a veces con límites o datos públicos) y planes de pago para escalar.

De pago

Plataforma comercial. Suele haber prueba o plan mínimo, pero el uso serio se cotiza con ventas.

Auto-etiquetado

Su foco es etiquetar automáticamente con IA en vez de hacerlo a mano caja por caja.

03 ¿Por dónde empiezo?

Atajos según tu situación

“Solo quiero aprender y probar.”
makesense.ai para anotar en el navegador sin instalar nada, o el plan gratis de Roboflow.
“Quiero un proyecto completo: anotar → entrenar → desplegar.”
Roboflow cubre todo el pipeline en un solo lugar.
“Quiero una herramienta de anotación potente y gratis, sin límites.”
CVAT auto-hospedada, o AnyLabeling si prefieres una app de escritorio.
“Tengo muchas imágenes de objetos comunes y no quiero etiquetar a mano.”
Autodistill etiqueta automáticamente con modelos fundacionales.
“Mi proyecto mezcla texto + imagen + audio.”
Label Studio es el más flexible para datos multimodales.
“Soy una empresa: necesito calidad y QA a escala.”
SuperAnnotate, Labelbox, V7 o Encord (este último, fuerte en imagen médica).
