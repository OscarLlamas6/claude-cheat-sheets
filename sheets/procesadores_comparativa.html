Entiende qué hace cada chip, cómo se diferencian y cuándo se usa cada uno.

🧠
Unidad de proceso central
CPU
Central Processing Unit
El cerebro general del ordenador. Ejecuta instrucciones de forma secuencial y es excelente resolviendo tareas complejas y variadas una a la vez.
🎮
Unidad de proceso gráfico
GPU
Graphics Processing Unit
Diseñada para procesar miles de tareas simples en paralelo. Nació para gráficos, pero hoy también potencia la IA y el cómputo científico.
🤖
Unidad de proceso neuronal
NPU
Neural Processing Unit
Especializada exclusivamente en operaciones de inteligencia artificial y aprendizaje automático. Consume muy poca energía para tareas de IA.
⚡
Unidad de proceso acelerada
APU
Accelerated Processing Unit
Combina CPU y GPU en un mismo chip. Ideal para dispositivos compactos donde se busca un balance entre potencia y eficiencia energética.
🔬
Unidad de proceso tensorial
TPU
Tensor Processing Unit
ASIC creado por Google exclusivamente para operaciones de tensores en Machine Learning. Potencia modelos masivos como Gemini en la nube de Google.

¿Cuándo usar cada uno?

🧠 CPU

  • Navegar por internet y usar aplicaciones
  • Programar y compilar código
  • Gestionar bases de datos
  • Cualquier tarea del día a día
  • Servidores web y aplicaciones empresariales

🎮 GPU

  • Videojuegos de alta resolución
  • Entrenar modelos de IA / Deep Learning
  • Renderizado 3D y efectos visuales
  • Minería de criptomonedas
  • Simulaciones científicas masivas

🤖 NPU

  • Reconocimiento facial en tiempo real
  • Asistentes de voz (Siri, Cortana)
  • Traducción automática offline
  • Cámara inteligente en smartphones
  • IA local en laptops Copilot+ PC

⚡ APU

  • Laptops delgadas y livianas
  • Consolas portátiles (Steam Deck)
  • Juegos en resolución media sin GPU extra
  • Diseño gráfico casual
  • Computadoras compactas (mini PCs)

🔬 TPU

  • Entrenar modelos de lenguaje masivos (LLMs)
  • Inferencia de IA a gran escala en la nube
  • Investigación en Deep Learning
  • Google Search, YouTube, Gmail (IA interna)
  • Entrenamiento de Gemini, BERT, PaLM
