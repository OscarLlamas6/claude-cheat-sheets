~/.tmux.conf — Configuración recomendada
cambiar prefix a Ctrl+a (como screen)
unbind C-b  |  set -g prefix C-a  |  bind C-a send-prefix
numeración desde 1
set -g base-index 1  |  setw -g pane-base-index 1
mouse support
set -g mouse on
vi keys en copy mode
setw -g mode-keys vi
dividir con | y -
bind | split-window -h  |  bind - split-window -v
recargar config con r
bind r source-file ~/.tmux.conf \; display "Config recargada"
historial largo
set -g history-limit 50000
colores 256 / true color
set -g default-terminal "tmux-256color"
